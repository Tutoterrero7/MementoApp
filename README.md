# Memento · recuerda lo importante

**Memento** es una app de tareas y recordatorios con sincronización en la nube, calendario, notificaciones y estadísticas de productividad.

🌐 **Landing page:** [tutoterrero7.github.io/MementoApp](https://tutoterrero7.github.io/MementoApp/)

## 📥 Descargar APK

[⬇️ Descargar última versión (v1.0)](https://github.com/Tutoterrero7/MementoApp/releases/download/v1/Memento.apk)

*Requisito: Android 8.0+*

## ✨ Funcionalidades

- Tareas con prioridad, fecha/hora y estados (Pendiente / En progreso / Completada)
- Categorías personalizables con colores
- Filtros, ordenación y búsqueda
- Calendario interactivo
- Notificaciones automáticas
- Sincronización en la nube (Supabase) y copias de seguridad
- Estadísticas de productividad semanal/mensual

## 🛠️ Stack

- Kotlin · Android Studio · Jetpack Compose
- MVVM + Clean Architecture
- Room (local) + Supabase (PostgreSQL)
- Dagger Hilt · Coroutines · Navigation Component
